Important FAQ

What is the DOM in JavaScript?

The DOM (Document Object Model) represents a webpage's structure as a tree of objects. JavaScript interacts with the DOM to dynamically update content, styles, and elements on the page.

How do you select elements from the DOM using JavaScript?

You can select DOM elements using methods like getElementById, getElementsByClassName, querySelector, or querySelectorAll to find elements based on their ID, class, tag, or CSS selector.

What is an event handler in JavaScript?

An event handler is a function triggered by an event (e.g., a click). Use addEventListener() to attach event handlers to elements, allowing you to handle user interactions like clicks or form submissions.

What is event delegation?

Event delegation assigns a single event listener to a parent element, handling events on its child elements. It optimizes performance by reducing memory usage and leveraging event bubbling to capture events at a higher level.

What is event bubbling, and how can you prevent it?

Event bubbling propagates events from a child element to its parents. Use event.stopPropagation() to prevent an event from bubbling up, stopping it from triggering handlers on parent elements.
